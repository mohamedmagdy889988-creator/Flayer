# Flayer

A modern, responsive front-end web project featuring a grid-based layout showcasing Apple products and technology innovations.

## Project Overview

This project is an assignment that demonstrates proficiency in HTML, CSS, and responsive web design. It showcases various Apple products including iPhone 6S, keyboards, and iMac with elegant grid layouts and multimedia content.

## Features

- **Responsive Design**: Fully responsive layout that works on all device sizes
- **Grid Layout**: Utilizes CSS Grid for modern layout structure
- **Product Showcase**: Displays Apple products with descriptions and call-to-action links
- **Font Integration**: Uses Google Fonts (Arvo, Montserrat, Open Sans) for typography
- **Image Assets**: Includes high-quality product images and graphics
- **Icon Library**: Integrated Font Awesome icons (all.min.css)

## Project Structure

```
flayer/
├── index.html          # Main HTML file
├── css/
│   ├── all.min.css     # Font Awesome icons
│   ├── style.css       # Main stylesheet
│   └── media.css       # Responsive media queries
├── images/             # Product and brand images
├── webfonts/           # Font files
└── README.md           # Project documentation
```

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Grid layout
- **Responsive Design**: Mobile-first approach with media queries
- **Google Fonts**: Web typography
- **Font Awesome**: Icon library

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/flayer.git
```

2. Navigate to the project directory:
```bash
cd flayer
```

3. Open the project in your browser:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

Or simply double-click the `index.html` file in your file explorer.

## Usage

The website is fully functional and ready to view. Simply open `index.html` in any modern web browser to see the responsive grid layout in action.

## Responsive Features

The project includes media queries in `css/media.css` to ensure optimal viewing experience on:
- Desktop browsers (1024px and above)
- Tablets (768px to 1023px)
- Mobile devices (below 768px)
